LeafScript 的解譯器會先讀取原始碼，接著透過 Lexer 將程式碼切成 Token。
例如：
let x = 1;
會變成：
("ID", "let")
("ID", "x")
("OP", "=")
("NUMBER", "1")
("SEMI", ";")
接著 Interpreter 會逐一讀取 Token。
當它讀到 let 時，就知道這是一個變數宣告。
let x = 1;
會被解譯成：
variables["x"] = 1
當它讀到 print 時，就會輸出資料。
print sum;
會被解譯成：
print(variables["sum"])
當它讀到 while 時，會先判斷條件是否成立。
while x <= 5
如果條件成立，就執行大括號內的程式。
如果條件不成立，就跳出迴圈。
